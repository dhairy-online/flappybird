# flappybird_deno

flappybird clone of written in
[deno_sdl2](https://github.com/littledivy/deno_sdl2)

## Installation
 
```shell
$ git clone https://github.com/dhairy-online/flappybird.git
$ cd flappybird/
$ deno compile -A --unstable main.js
$ ./flappybird
```
